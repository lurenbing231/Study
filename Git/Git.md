   ## 一、安装及配置
   ### Git安装
   Git从网上下载就行，我的电脑是win10，一路傻瓜式操作就安装完了。
   
   ### Git用户名及邮箱配置
   用户名
   ```
   git config --global user.name "zhaoliang"
   ```
   用户邮箱
   ```
   git config --global user.email "lzhao*****@163.com"
   ```
   配置好后用以下命令查看
   ```
   git config user.name
   git config user.email
   ```
